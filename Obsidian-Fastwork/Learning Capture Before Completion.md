# Learning Capture Before Completion

เมื่อ Agent ตั้งสมมติฐานผิด ถูกผู้ใช้แก้ ทำให้เกิด regression/rework เจอ failure ที่ไม่ชัดเจน
หรือค้นพบวิธีป้องกันที่ใช้ซ้ำได้ งานยังไม่ถือว่าเสร็จจนกว่าจะบันทึกบทเรียนไว้ให้รอบถัดไป

- ค้นหาโน้ตเดิมก่อน ถ้ามี owner ของแนวคิดนี้อยู่แล้วให้อัปเดตแทนการสร้างซ้ำ
- ถ้ายังไม่มี ให้สร้างหนึ่งโน้ตต่อหนึ่งบทเรียนและเชื่อมจาก index ที่เกี่ยวข้อง
- บันทึกอาการ สาเหตุที่ยืนยันแล้ว กฎป้องกัน และวิธีตรวจสอบ
- hypothesis ที่ยังไม่ยืนยันให้อยู่ใน Open Work Ledger ไม่ยกระดับเป็นข้อเท็จจริงถาวร
- ข้ามได้เฉพาะ typo หรือ noise ครั้งเดียวที่ไม่มีบทเรียนใช้ซ้ำ
